# 每日法律雷達 Remotion 影片

75 秒、1920 × 1080、30 fps 的講座影片，說明法律資訊如何從官方來源進入工作台，經過去重、摘要與律師確認後才對外使用。

## 指令

```bash
npm install
npm run dev
npm run still
npm run render
```

## Composition

- ID：`LegalRadar`
- 程式：`src/Composition.tsx`
- 完整影片：`out/legal-radar.mp4`
- 預覽圖：`out/legal-radar-preview.png`

## 使用原則

- 所有動畫由 Remotion frame 驅動。
- 影片無旁白，適合作為講座開場或章節轉場。
- 官方來源、AI 摘要與律師審核被清楚分開。
